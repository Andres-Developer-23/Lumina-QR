# Lumina QR - Plan de Negocios

## Descripción

Lumina QR es un emprendimiento dedicado a la creación de velas artesanales únicas que combinan la artesanía tradicional con la tecnología moderna. Cada vela viene acompañada de un código QR que revela la historia detrás de su creación, ofreciendo una experiencia interactiva y personalizada.

Este proyecto es una página web que presenta el plan de negocios de Lumina QR, mostrando nuestros productos, proceso artesanal, testimonios de clientes y información de contacto.

## Características

- **Velas Artesanales**: Productos hechos a mano con materiales ecológicos
- **Tecnología QR**: Cada vela incluye un código QR único con información personalizada
- **Sostenibilidad**: Compromiso con materiales biodegradables y procesos ecoamigables
- **Variedad de Aromas**: Líneas especializadas para relajación, energía, amor, serenidad, pureza, alegría, armonía y sueños
- **Diseño Responsivo**: Página web optimizada para todos los dispositivos

## Tecnologías Utilizadas

- **HTML5**: Estructura de la página web
- **CSS3**: Estilos y diseño visual
- **JavaScript**: Interactividad y funcionalidades dinámicas
- **Bootstrap 5**: Framework CSS para diseño responsivo
- **Font Awesome**: Iconos vectoriales
- **Google Fonts**: Tipografía Montserrat

## Instalación y Uso

### Opción 1: Ver en línea
Visita el repositorio en GitHub y navega a la pestaña "Pages" para ver la versión en vivo del sitio web.

### Opción 2: Ejecutar localmente
1. Clona este repositorio:
   ```bash
   git clone https://github.com/Andres-Developer-23/Lumina-QR.git
   ```

2. Navega al directorio del proyecto:
   ```bash
   cd Lumina-QR
   ```

3. Abre el archivo `index.html` en tu navegador web preferido.
